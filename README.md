# Testing
 
Webpack:
  
* Run the command webpack ./test/entry.js ./test/bundle.js. 
* Open ./test/index.html in your browser.

In browser:  
* Open ./test/vanilla.html

Through npm:

* npm install idle.js --save
* In node_modules/idle-js you will have access to library.
